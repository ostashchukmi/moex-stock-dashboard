# Разработка информационно-аналитической системы для исследования закономерностей поведения рынка акций Московской биржи

Интерактивный дашборд, позволяющую пользователям получить доступ к результатам исследования закономерностей поведения рынка акций Московской биржи.

## Структура проекта

🔍 research.ipynb блокнот с исследованием: загрузка данных, подготовка и очистка данных, подготовка признаков для последующего применения алгоритмов кластеризации, разведочный анализ, применение моделей кластеризации, оценка качества моделей и их применимости к рассматриваемому набору данных, выбор наиболее релевантной модели и интерпретация результатов;<br>
⚙️ app.py создает объект Dash-приложения, подключает внешний стиль, задает заголовок страницы, формирует интерфейс приложения и регистрирует функции обратного вызова;<br>
⚙️ layouts.py структура пользовательского интерфейса дашборда;<br>
⚙️ graphs.py построение графиков и таблицы, которые отображаются в дашборде;<br>
⚙️ callbacks.py интерактивность дашборда;<br>
⚙️ seed.py скрипт для наполнения базы данных;<br>
⚙️ dictionary.xlsx справочник информации об акциях, используется seed.py для наполнения базы данных;<br>
⚙️ requirements.txt список Python-библиотек, необходимых для работы проекта;<br>
⚙️ utils/models.py описывает структуру таблиц PostgreSQL на уровне Python-классов;<br>
⚙️ utils/db.py подключение приложения к базе данных PostgreSQL и загрузку данных из таблиц;<br>
⚙️ assets/custom.css внешний вид дашборда, хранит каскадную таблицу стилей;
